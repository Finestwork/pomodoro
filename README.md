# Installed Packages
- PostCSS
    - Autoprefixer
    - CSSNano for minimizing & optimizing css files.
    - PurgeCSS for removing unused CSS.

- Sass as a pre-processor for html and css.
- Normalize.css
- Vite as a bundler tool.

# Installation
- run `npm install`
- run `npm run dev` or `npm run build`.
